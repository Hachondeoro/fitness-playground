## Getting Started

### Install the dependencies:
```bash
    npm run install
```

### Running the development server.

```bash
    npm run dev
```
Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Running the production server.

```bash
    npm run build
```
```bash
    npm run start
```
